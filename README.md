# SpotMicroAI

여기서는 Raspberry Pi에 MG996R 서보 모터와 각종 센서로 SpotMicroAI를 구현한다. 

* https://gitlab.com/custom_robots/spotmicroai 를 참고해서 구현 중이다. 마지막 업데이트가 2년 전이라 현재와 라이브러리 등은 수정이 필요하다.
